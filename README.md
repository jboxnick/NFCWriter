# NFCWriter
Tutorial - How to write simple String on NFC tag using iOS device and Swift 5 - CoreNFC

1. Add Near Field Communication Tag Reading to your Signing & Capabilities
2. Add import CoreNFC to the top of your VC file
3. Add NFCNDEFReaderSessionDelegate to your VC and add the needed delegate functions
4. Configure the delegate functions as wanted

// Sample project shows how to write a simple string on a NFC tag


Deployment target iOS 13.0 or higher

ONLY WORKS ON PHYSICAL DEVICE, NOT ON SIMULATOR!
